우조향 약함 점검 체크리스트

1
서보혼 물리 중립 확인
PWM 0.116로 설정했을 때 서보혼이 실제로 정중앙인지 자로 측정
2
SERVO_CENTER 재캘리브레이션
실제 중립이 0.116보다 높으면 → CENTER 값을 올려야 좌우 대칭
3
우조향 시 물리 저항 확인
링키지·타이로드가 우측에서 기계적으로 걸리는지 체크
4
하드웨어 PWM 전환 (근본 해결)
소프트웨어 PWM 타이밍 불안정 → config.txt에
dtoverlay=pwm,pin=17,func=2 추가
