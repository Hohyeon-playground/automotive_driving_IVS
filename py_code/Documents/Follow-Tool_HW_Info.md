## 3. 서보모터 (MG996R)

### 3-1. 배선 - 보세요

```
MG996R 서보모터
┌─────────────────────┐
│  빨강 (VCC)  ───────┼──────────► 라즈베리파이 2번 핀 (5V)
│  갈색 (GND)  ───────┼──────────► 공통 GND
│  노랑 (Signal)──────┼──────────► GPIO 17 (물리적 11번 핀)
└─────────────────────┘
```

### 3-2. 소프트웨어 설정값 -> 걍 몰라도 됨. cluade 시키세요

| 항목 | 값 |
|---|---|
| 제어 방식 | gpiozero PWMOutputDevice + LGPIOFactory |
| GPIO 핀 | **GPIO 17 (물리적 11번 핀)** |
| PWM 주파수 | 50Hz |
| SERVO_CENTER | 0.071 (1.425ms) |
| SERVO_LEFT | 0.025 (0.5ms) |
| SERVO_RIGHT | 0.118 (2.35ms) |

### 3-3. 조향 매핑 -> 이건 필요할 수 있음

```
angle 입력값 → gpiozero value → 실제 조향

  -1.0  ────────  0.025 ───────  최대 좌조향
  -0.5  ────────  0.048 ───────  좌조향
   0.0  ────────  0.071 ───────  중립 (직진)
  +0.5  ────────  0.095 ───────  우조향
  +1.0  ────────  0.118 ───────  최대 우조향

  ◄────────────────────────────────────►
  좌                 중립               우
```

## 5. 실행 시 필수 명령어

```bash
# 1. 카메라 점유 해제
sudo fuser -k /dev/video0 2>/dev/null
sudo fuser -k /dev/video2 2>/dev/null

# 2. ROS2 환경 설정
source /opt/ros/jazzy/setup.bash
source ~/ros2_ws/install/setup.bash

# 3. 노드 실행 순서 (터미널 4개)
ros2 run motor_node motor_node          # 터미널 1
ros2 run aruco_publisher aruco_node     # 터미널 2
ros2 launch hand_gesture hand_gesture.launch.py camera_index:=2  # 터미널 3
ros2 run state_machine_node state_machine_node  # 터미널 4
```

## 11. /boot/firmware/config.txt PWM 설정

```bash
# 현재 설정 (GPIO 12, 13 하드웨어 PWM)
dtoverlay=pwm-2chan,pin=12,func=4,pin2=13,func2=4
```

> ⚠️ GPIO 18 (서보)은 현재 **소프트웨어 PWM** 사용 중 → 하드웨어 PWM 전환 필요

하드웨어 PWM 전환 시 config.txt에 추가:
```bash
dtoverlay=pwm,pin=18,func=2
```
